# Profile Image Placeholder

Place your profile photo here and update the `index.html` file to reference it.

## Instructions

1. Add your image to this folder (e.g., `profile.jpg`, `profile.png`)
2. Open `index.html`
3. Find the hero section (around line 160)
4. Replace the placeholder URL with your image path:

```html
<img src="assets/images/profile.jpg" alt="Andrew Fernandes - Profile Photo">
```

## Recommended Specifications

- **Size**: 350x350 pixels (square)
- **Format**: JPG or PNG (WebP for better compression)
- **Quality**: High enough for 2x displays (700x700 source)
- **File size**: Under 100KB for fast loading
